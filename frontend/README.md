# NeoxMeet Frontend

Next.js client for NeoxMeet. The register page is wired to the backend authentication API.

## Environment

Set API base URL for backend communication:

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:3001
```

Create `.env.local` from `.env.example` before starting the app.

## Running locally

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```
